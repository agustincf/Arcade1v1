# apps/web — El sitio web (frontend)

Lo que ve y toca el jugador:

- **Home** con cards de juegos (Ajedrez, Flappy 1v1).
- **Seleccion de mesa** (1 / 5 / 10 / 20 USDC).
- **Conexion de billetera** cripto.
- **Sala de emparejamiento** (matchmaking).
- **Interfaz de cada juego** con marcador y resultado.
