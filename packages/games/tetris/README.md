# games/tetris — Tetris (asincronico, por puntaje)

Modulo del juego Tetris clasico. Cada jugador juega su propio intento ("run")
y gana el que hace **mas puntos**.

- Asincronico: no hace falta que jueguen al mismo tiempo. Cada uno juega dentro
  de la ventana de la partida (ej: 1 hora desde que se emparejan).
- Cumple el contrato de `packages/game-sdk` (modelo por puntaje).
